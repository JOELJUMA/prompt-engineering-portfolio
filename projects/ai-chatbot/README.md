# 🤖 AI Chatbot (Prompt-Based)

##  Overview
This project demonstrates a prompt-engineered chatbot designed to provide structured, helpful, and context-aware responses.

##  Objective
To design prompts that improve response quality, tone, and usefulness in conversations.

## Prompt Design

### Base Prompt
"Answer the user's question."

### Improved Prompt
"Act as a helpful and intelligent assistant. Answer the user's question clearly and concisely. If applicable, provide examples and actionable steps."

##  Example Interaction

### User:
"How can I improve my coding skills?"

### AI Output:
- Practice daily with small projects  
- Learn by building real applications  
- Read and understand other people's code  
- Use online platforms for challenges  

## 🔁 Iteration Process
- Initial responses were vague  
- Added structure and role prompting  
- Improved clarity and usefulness  

## ✅ Outcome
More structured, actionable, and user-friendly responses
